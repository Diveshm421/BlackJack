# BlackJack
This is a simple console-based Blackjack game written in Java.

## How to Play
- The player starts with coins
- You can bet coins each round
- You can hit or stand
- The dealer follows standard Blackjack rules
- Coins carry over between rounds

## How to Run
Download `BlackJack.jar` from:
https://github.com/Diveshm421/BlackJack/releases/tag/v1.0
Run it using:
```bash
java -jar BlackJack.jar
